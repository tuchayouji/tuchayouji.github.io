# Tucha的个人博客

记录动漫、代码与日常随笔的个人博客。

## 技术栈

- [Astro](https://astro.build/)
- [Mizuki主题](https://github.com/LyraVoid/Mizuki)

## 功能

- 追番记录
- 日记功能
- 相册展示
- 技能展示
- 时间线
- 项目展示
- 友链管理

## 本地运行

```bash
pnpm install
pnpm dev
```

## 构建部署

```bash
pnpm build
```

构建产物在 `dist` 目录，推送至GitHub后通过GitHub Actions自动部署到GitHub Pages。
